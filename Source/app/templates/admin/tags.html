    How Tags Work

    • Tags are hierarchical — parent tags group related child tags (e.g., Hardware › Docking Station).
    • Add tags to tickets to categorize issues. Use the Tags section on any ticket.
    • Add tags to assets — when a ticket is created for that asset, tags are auto-applied.
    • Child tags inherit their parent's color unless you assign a specific color.
    • Drag rows to reorder within a level. Drop a root tag onto a sub-area to nest it.
    • Tags used on tickets or assets cannot be deleted until removed from all tickets/assets.
